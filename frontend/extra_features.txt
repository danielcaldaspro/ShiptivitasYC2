Beyond the core requirements, I implemented several high-end features to demonstrate advanced engineering skills, including a sophisticated Glassmorphism UI system, a centralized state architecture (Lifting State Up) to synchronize a global progress counter in the Navbar, and an advanced 4-mode smart filter (A-Z, Z-A, and Priority-based). Additionally, I developed a high-performance 60 FPS physics-based magnetic mascot using direct DOM manipulation and integrated a responsive design with fixed-height viewport columns for a native-app feel, while upgrading the entire project to React 18 for maximum efficiency.
